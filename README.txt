# GitHubs-Actions Project

Este projeto é um servidor Node.js que expõe a porta 3000. 
Ele está configurado para integração contínua e entrega contínua (CI/CD) usando GitHub Actions, incluindo a construção de uma imagem Docker e o deploy para o DockerHub.
